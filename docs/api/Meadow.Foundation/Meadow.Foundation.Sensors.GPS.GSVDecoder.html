Class GSVDecoder

Process the Satellites in view messages from a GPS module.

Inheritance
System.Object
NMEADecoder

Namespace: Meadow.Foundation.Sensors.GPS
Assembly: Sensors.GPS.NMEA.dll

public class GSVDecoder : NMEADecoder
Remarks

The satellites in view messages can contain multiple packets one for each satelite. There can also be multiple messages making up the total list of satelites. This class brings all of the messages together in a single message for the consumer.
